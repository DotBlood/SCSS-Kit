Абзацы:

P - Заголовок в header.

P - предназначен для заголовков блоков.

P - предназначен для подзаголовков блоков.

P с классом roboto - текст для заголовка страницы, он имеет 2 класса: `text - стандартный класс текста, roboto - это название шрифта`.

P с классом open-sans - текст для заголовка страницы, он имеет 2 класса: `text - стандартный класс текста, open-sans - это название шрифта`.

P - стандартный класс для текста.

Как вы могли заметить для текста по умолчанию не идёт какой-то определённый шрифт, поэтому его нужно указывать отдельно. Также можно поместить класс open-sans или roboto в body для того, чтобы определить, какой именно будет использоваться шрифт по умолчанию, или это можно делать отдельно, через добавление данных атрибутов непосредственно на сам текст.
